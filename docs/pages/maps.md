# Maps

The maps page is a dedicated page in the web app. It is the Cape Town gallery map.

## Base map

The map uses Google Maps as its underlying base layer, scoped and centered on Cape
Town.

## Galleries as nodes

The map is populated with real Cape Town art galleries, rendered as nodes on the map.

Only galleries, which are physical places, are rendered as nodes. Events, posts, and creatives
without a physical location are not nodes. They live in the feed and cross-link to a gallery
node when relevant.

## Stripped interface

Extraneous map interface is stripped away. Standard Google Maps points of interest,
labels, and controls that are not relevant are removed, leaving a clean custom-styled
map focused on the galleries.

The map stays a real, legible map. It is professional, sleek, and minimal, rendering only the
information we need. It is not abstracted, desaturated, or turned black and white.

## Gallery card

Selecting a node, or a node coming into focus, renders a panel for that gallery beside the map.
The panel presents that gallery's information and images. The gallery information shows when the
node is in focus or clicked.

Arriving from a gallery account card in the feed carousel opens the map with that gallery's node
already focused and its panel in view. The account card itself does not expand; it routes here
instead.

## Intent

The maps page shows recent, regularly refreshed information and images. This is a core
differentiator. Where Google Maps shows outdated information, Grounded Art keeps gallery
details current. Where social media keeps each gallery fragmented across separate
accounts, Grounded Art holds every gallery, event, and post in one space.
